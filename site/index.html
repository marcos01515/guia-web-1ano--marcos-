O que são HTML, CSS e JavaScript?

HTML

Estrutura o conteúdo da página web, usando tags para definir títulos, parágrafos, imagens, links, etc.

CSS

Define o estilo visual do HTML, como cores, fontes, espaçamento e responsividade.

JavaScript

Adiciona interatividade e dinamismo, manipulando HTML/CSS, respondendo a eventos e persistindo dados no navegador (ex: localStorage).

Boas Práticas Profissionais

  • Acessibilidade: Facilitar o uso por todos, incluindo pessoas com deficiência.
  • Semântica: Usar tags HTML apropriadas para dar significado ao conteúdo.
  • Responsividade: Garantir boa visualização em diferentes dispositivos usando CSS.
  • Organização: Separar arquivos, nomear claramente, comentar o código e manter estrutura lógica.
  • Versionamento: Utilizar sistemas como Git para controlar alterações e facilitar colaboração.

Panorama de Tecnologias Web

  • Front-end: Interface e interação do usuário (HTML, CSS, JS).
  • Back-end: Lógica de servidor e banco de dados (não permitido neste projeto).
  • Bancos de dados: Armazenamento de dados (usar apenas localStorage).
  • DevOps: Automação, deploy e manutenção do site.
  • Testes: Garantia de qualidade do código.

Como escolher: Neste projeto, usar apenas HTML5, CSS3 e JavaScript ES6+, sem frameworks e sem back-end.

Fluxograma de um Projeto Web Completo

  1. Ideia: Definir objetivo e funcionalidades.
  2. Planejamento: Esboço do layout e recursos.
  3. Design: Criação do visual.
  4. Desenvolvimento: HTML (estrutura), CSS (estilo), JS (interatividade).
  5. Testes: Verificar funcionamento e responsividade.
  6. Ajustes: Corrigir e melhorar.
  7. Deploy: Publicar o site.
