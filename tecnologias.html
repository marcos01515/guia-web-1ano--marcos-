Panorama de Tecnologias Web

  • Front-end: Interface e interação do usuário (HTML, CSS, JS).
  • Back-end: Lógica de servidor e banco de dados (não permitido neste projeto).
  • Bancos de dados: Armazenamento de dados (usar apenas localStorage).
  • DevOps: Automação, deploy e manutenção do site.
  • Testes: Garantia de qualidade do código.

Como escolher: Neste projeto, usar apenas HTML5, CSS3 e JavaScript ES6+, sem frameworks e sem back-end.
