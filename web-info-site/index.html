1. O que são HTML, CSS e JavaScript?

  • HTML: Linguagem de marcação usada para estruturar o conteúdo de páginas web.
  • CSS: Linguagem de estilos utilizada para definir a aparência visual do conteúdo HTML.
  • JavaScript: Linguagem de programação que permite criar interatividade e dinamicidade nas páginas web.

Papel de cada um: HTML constrói a base, CSS estiliza, JavaScript traz funcionalidades.

2. Boas práticas profissionais

  • Acessibilidade: Tornar o site utilizável por todos, incluindo pessoas com deficiência.
  • Semântica: Utilizar tags HTML apropriadas para dar significado ao conteúdo.
  • Responsividade: Garantir que o site funcione bem em diferentes dispositivos e tamanhos de tela.
  • Organização: Manter o código e os arquivos bem estruturados e documentados.
  • Versionamento: Utilizar sistemas como Git para controlar alterações e colaborar em equipe.

3. Panorama de tecnologias web

  • Front-end: Interface do usuário (HTML, CSS, JavaScript, frameworks como React, Vue).
  • Back-end: Lógica de negócio e acesso a dados (Node.js, Python, Java, PHP).
  • Bancos de dados: Armazenamento de informações (MySQL, MongoDB, PostgreSQL).
  • DevOps: Automação de deploy, integração contínua, infraestrutura (Docker, CI/CD).
  • Testes: Garantia de qualidade (Jest, Mocha, Selenium).

Como escolher? Avalie requisitos do projeto, equipe, escalabilidade e comunidade das tecnologias.
